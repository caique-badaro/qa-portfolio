Análise de mapas de calor

Compreendendo o comportamento do usuário

A análise de mapas de calor realizada na página inicial do site Apoio Entrega (2022) forneceu insights valiosos sobre onde os usuários demonstram maior engajamento e interesse. Com base nesses dados, foi possível otimizar o design da página para melhorar a experiência do usuário (UX) e atingir os objetivos do projeto.

Mapas de Calor são ferramentas essenciais para análise de interação do usuário em sites e aplicativos. Eles representam, de forma visual, padrões de comportamento, como cliques, movimento do mouse e scroll (rolagem), utilizando cores para destacar áreas com maior ou menor atividade.

Pesquisa com usuários

Insights para otimização do site Apoio Entrega

Durante o desenvolvimento do projeto, foi realizada uma pesquisa on-line com a base de clientes do site Apoio Entrega. O principal objetivo foi compreender a percepção dos usuários sobre promoções, ferramentas e áreas de maior interesse disponíveis na plataforma.

A pesquisa também desempenhou um papel essencial ao identificar funcionalidades e recursos desejados pelos usuários, oferecendo insights valiosos para otimizar a experiência do cliente e alinhar o site às suas expectativas.

A pesquisa foi estruturada em três etapas, detalhadas a seguir:

Etapa 1

Matriz CSD

Utilizada para identificar os elementos-chave e organizar de forma sistemática o conhecimento existente.

Etapa 2

Hipóteses

Definição de hipóteses e suposições fundamentadas a partir dos dados extraídos na Matriz CSD.

Etapa 3

Pesquisa

Construção do formulário, seleção da base, acompanhamento e análise.

Benchmarking

Realizamos uma análise detalhada de benchmarking para comparar as práticas adotadas por concorrentes do setor. Essa avaliação forneceu insights valiosos sobre estratégias eficazes e oportunidades para aprimorar nossa plataforma.

O benchmarking é reconhecido como uma estratégia essencial para identificar tendências, implementar melhorias e fortalecer a competitividade de um projeto no mercado.

Durante o desenvolvimento, foram seguidos os passos abaixo:
Identificação dos concorrentes

Principais práticas e empresas que atuam no segmento.

Análise dos sites, desktop e mobile

Identificar boas práticas, pontos fortes e estratégias para melhorar a experiência de navegação.

Mapeamento das informações

Organização estratégica das informações coletadas.

Referência criativa projeto

Documentação das principais referências mapeadas.

Informações complementares

Sobre o projeto, processos e ferramentas utilizadas

Qual o principal objetivo do projeto?

Aprimorar a percepção do site como uma plataforma de preços competitivos e promoções impactantes.

Quais foram os principais desafios identificados

Baixa visibilidade para as ofertas do site e falta de distinção clara entre produtos promocionados e ofertas abertas para o público geral.

Como a proposta de revisão abordou esses desafios?

Foi realizada uma intervenção para destacar as principais promoções na primeira seção da página, proporcionando maior visibilidade aos produtos com preços atrativos.

Houve a ampliação da área de exibição dos produtos do folheto otimizando um comportamento mapeado durante a análise da jornada dos usuários. Reforçando a percepção do Apoio Entrega como uma plataforma de preços competitivos e promoções impactantes.

Houve pesquisa de usuário envolvida no processo de revisão?

Sim, foi realizada uma pesquisa em conjunto com a Matriz CSD para validar as expectativas e hábitos dos usuários.

Quais métricas foram usadas para avaliar os resultados da revisão?

Para avaliar o resultado das mudanças foi utilizada a pesquisa de CSAT ao final das compras e o monitoramento dos mapas de calor.

Onde e como atua o Apoio Entrega?

O Apoio Mineiro é uma rede de supermercados do Grupo Supernosso, posicionado por toda Belo Horizonte (MG) e Região Metropolitana.

O Apoio Mineiro atua com vendas em loja física e plataformas digitais, no ambiente digital utiliza o nome Apoio Entrega, oferencendo opções diversificas para os seus clientes abastecerem seus lares e negócios.
